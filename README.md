# One For All

[![luacheck](https://github.com/mt-mods/one_for_all/actions/workflows/luacheck.yml/badge.svg)](https://github.com/mt-mods/one_for_all/actions/workflows/luacheck.yml?query=branch%3Amaster)

This simple mod causes every player to die if any one dies,
ideally forcing everyone to work together.
There are no mod dependencies for this mod.

The source is available on [github](https://github.com/mt-mods/one_for_all).

Code: LGPL2
